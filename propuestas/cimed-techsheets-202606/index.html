Sistema de fichas técnicas digitales para el catálogo CIMED

Esta propuesta presenta un sistema de documentación técnica para el catálogo de partes eléctricas y equipos de medición de CIMED. El objetivo es reemplazar los Tech Sheets actuales por fichas digitales con identidad de marca consistente, aptas para consulta web y exportación a PDF sin pérdida de calidad.

Incluye: marco de identidad visual (paleta, tipografías, usos del logo), ficha técnica estándar de 1 página, y un formato extendido de 2 páginas para productos de alta especificación. Todo el contenido es demostrativo y usa datos reales del catálogo para ilustrar el resultado final.
